# Daily Summaries

Generated daily summary files, organized by year.

## Structure

```
Daily/
└── YYYY/
    └── YYYY-MM-DD.md
```

Files are written here automatically by the summary generation workflow.
They are human-readable and can be browsed directly in Obsidian.
